1. 프로젝트 개요

제목: AWS Bedrock 기반 AI Agent형 도로교통법 검색 및 자동 분석 시스템

목표: RAG(Retrieval-Augmented Generation), AI Agent, MCP(Model Context Protocol)를 활용한 다중 데이터 소스 통합 검색 및 자동 분석 시스템 개발
일반 운전자와 시민 누구나 자연어로 질문해도 최신 도로교통법·시행규칙을 즉시 확인할 수 있는 대국민 서비스를 제공하며,
AWS Bedrock의 Knowledge Base에 도로교통법, 행정해석, 판례 등의 PDF 자료를 적재해 신뢰도 높은 지식 그래운딩을 보장합니다.
사용자의 자연어 질의를 이해하고, 여러 데이터 소스(문서, 데이터베이스, API 등)에서 관련 정보를 검색하여 통합된 답변을 생성하는 지능형 시스템을 개발하는 것을 목표로 합니다.

3. 목표 및 내용
## 개발목표 : 개발하고자 하는 시스템의 목표
"""
- RAG 기반의 정확한 정보 검색 및 생성 시스템
- 다중 Agent를 활용한 복잡한 질의 처리 및 자율적 문제 해결
- MCP를 통한 확장 가능한 외부 도구 통합 아키텍처

## 개발내용 : 구성과 기능, 특징 등
"""
- 문서 전처리 및 벡터 데이터베이스 구축 : 다양한 형식의 문서(PDF, Word, Excel, 웹페이지) 
- 수집 및 파싱 : 텍스트 청킹(chunking) 및 임베딩 생성 및 Vector DB 구축

- AWS Bedrock Knowledge Base에 도로교통법·시행규칙 PDF와 행정안내서를 업로드하고, 자동 동기화 파이프라인을 구성해 최신 자료를 상시 반영
- 법령·판례·행정해석 문서를 버전 태깅하여 개정 이력과 효력 시점을 메타데이터로 저장
- RAG 파이프라인 구현 : 의미 기반 유사도 검색(Semantic Search), Advanced RAG (하이브리드 검색) 구현, 컨텍스트 기반 답변 생성
- AWS Bedrock 상의 Claude/Llama 모델에서 Function Calling을 사용해 인용·출처·근거 조항을 구조화된 JSON으로 출력
- AI Agent 시스템 개발 : 질의 분석 및 작업 계획 수립 Agent, 정보 검색 및 필터링 Agent, Agent 간 협업 및 워크플로우 관리
- 단속 시나리오별 벌점/벌금 계산 Agent, 관련 조항 추천 Agent 등을 추가하여 다중 에이전트 협업 체계 확장
- MCP 기반 외부 도구 연동 : 데이터베이스 쿼리 도구, 웹 검색 API 연동, 파일 시스템 접근
- 사용자 인터페이스 : 채팅 기반 질의응답 인터페이스
- 행정공무원·일반 사용자용 대시보드에서 답변 히스토리, 개정 타임라인, PDF 자동 요약 결과를 시각화
"""
